# 行为准则

English version: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

## 我们的承诺

Wallet Map 希望成为一个实用、尊重他人、以证据为中心的开源项目。

参与者应保持体面、建设性，并谨慎处理隐私敏感话题。

## 期望行为

- 尊重他人，表达直接但克制。
- 基于证据讨论想法。
- 将批评聚焦于代码、文档和行为。
- 不分享私有钱包数据、secrets 或个人信息。
- 尊重项目关于分析和安全的边界。

## 不可接受行为

- 骚扰或人身攻击。
- 发布私钥、助记词、API key 或私有钱包列表。
- 鼓励滥用、垃圾操作或规避平台规则。
- 将弱分析信号误称为身份事实。

## 执行

维护者可编辑、隐藏或移除违反本行为准则的评论和贡献。
