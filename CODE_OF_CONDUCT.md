# Code of Conduct

## Our Pledge

We want Wallet Map to be a practical, respectful, and evidence-focused open-source project.

Participants are expected to be considerate, constructive, and careful with privacy-sensitive topics.

## Expected Behavior

- Be respectful and direct.
- Discuss ideas with evidence.
- Keep criticism focused on code, docs, and behavior.
- Avoid sharing private wallet data, secrets, or personal information.
- Respect the project boundary around analysis and safety.

## Unacceptable Behavior

- Harassment or personal attacks.
- Publishing private keys, seed phrases, API keys, or private wallet lists.
- Encouraging abuse, spam, or evasion of platform rules.
- Misrepresenting weak analysis signals as proof of identity.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct.
